# Stack y convenciones

- HTML + CSS sin frameworks. Nada de React, Vue ni Tailwind.
- Mobile-first.
- Comentarios en castellano.
- La identidad visual (colores, tipografías, espaciados) está definida en `design.md`.
  La IA debe respetar esos tokens — no inventar valores propios.
- Las imágenes con texto alternativo siempre tienen `alt` no vacío.
- Los textos en castellano usan `<html lang="es">`.

# Cómo trabajar conmigo

- Si una petición es ambigua, prefiere preguntar antes que asumir.
- Cambios pequeños y enfocados. Un commit por cosa.
- Si tocas el sistema de diseño, edita `design.md` primero, no `style.css` directamente.

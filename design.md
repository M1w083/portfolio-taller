
# Vibrant Minimalist Developer Portfolio — Design System

## Visual Identity
A balance of rigorous engineering and creative intuition. The aesthetic is clean, dark, and high-performance, utilizing deep surfaces and vibrant neon accents to represent "Ordered Energy."

## Color Palette
- **Surface**: `#13131c` (Deep midnight for primary backgrounds)
- **Surface Bright**: `#393842` (Elevated elements and containers)
- **Primary**: `#5d5dff` (Electric indigo for key actions and highlights)
- **Secondary**: `#c1c1ff` (Soft lavender-blue for supporting elements)
- **Error**: `#ff5d5d` (Vibrant red for critical alerts)
- **On-Surface**: `#ffffff` (Pure white for primary readability)
- **On-Surface Variant**: `#a1a1aa` (Muted grey for secondary information)

## Typography
- **Primary Font**: Montserrat (Modern, geometric, and highly readable)
- **Display**: Bold/Extrabold weights for impactful headlines.
- **Body**: Regular/Medium weights for clean text blocks.
- **Label Mono**: Used for technical data, code snippets, and system metrics to emphasize the developer persona.

## Geometry & Spacing
- **Roundness**: `ROUND_EIGHT` (8px corner radius) for a professional yet modern feel.
- **Grid**: Standard 12-column grid with generous gutters to maintain a minimalist, airy layout.
- **Hierarchy**: Clear distinction between content sections using subtle borders (`white/10`) and depth-based shadows.

## Key Components
- **Navigation**: Fixed top bar with a glassmorphism effect (blur) and active state indicators.
- **Cards**: Dark containers with subtle borders and hover-states that utilize the primary glow.
- **Buttons**: High-contrast primary actions with soft transitions and scale interactions.
- **Timelines**: Vertical lines with glowing nodes for professional experience.
